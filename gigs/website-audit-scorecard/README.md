# Website Audit Scorecard

## Target buyer
Websites owners and freelancers supporting SMB clients who need fast diagnosis before charging for redesign work:
- solo freelancers
- small agencies
- local service businesses wanting immediate improvement priorities
- coaches and consultants with landing pages

## Problem this solves
Most owners know they need a website refresh but do not know which fixes will move leads the most. This mini-site provides a practical scoring framework with a priority list of 5 high-impact improvements.

## What is included
- Manual audit scorecard with 10 rubric sliders
- Total score out of 50 with performance tier
- Auto-generated top-priority fix list from low-scoring categories
- Mobile-responsive interface for sharing with clients on calls
- Branded email CTA to turn the scorecard into a lead capture action

## Customization ideas
- Tune slider labels and weights to your niche (law, healthcare, SaaS, ecommerce).
- Convert slider options to checkboxes for simpler buyer workflows.
- Add PDF export (print CSS) for client handoff packages.
- Replace generic fix list text with pre-approved, niche-specific recommendations.

## Suggested selling-price range
- Base scorecard template: **$59 – $109**
- Branded consulting version with custom rubric: **$149 – $239**
- Full service bundle with implementation guide: **$279 – $399**

## Deploy notes
1. Publish this folder as-is for static delivery.
2. Keep slider defaults aligned with your standard diagnostic framework.
3. Test all ranges on desktop and mobile to ensure input responsiveness.
4. If the customer uses the tool in a live call, run one full score and share the action list.
