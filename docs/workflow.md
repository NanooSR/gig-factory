# Rapid Gig Workflow\n\n## 1. Pick a package\n- Landing page\n- Mini app\n- Browser game\n\n## 2. Gather the client brief\n- business/project name\n- audience\n- goal\n- style/colors\n- examples\n- deadline\n\n## 3. Generate code with ChatGPT/Codex\nUse the prompts in `prompts/`.\n\n## 4. Test locally\n- verify mobile layout\n- verify buttons/forms\n- verify empty/error states\n\n## 5. Commit locally\n- save milestones in git\n\n## 6. Push to GitHub\n- one repo per client or per template\n\n## 7. Deploy with Vercel\n- connect GitHub repo or use CLI\n\n## 8. Deliver\n- send preview link\n- offer 1 revision\n- upsell support/extra pages\n
