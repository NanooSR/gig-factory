# Profit-First Gigs Shortlist (Pack 1)

## Why these 3 mini-sites were chosen over browser games and generic templates

### 1) Higher purchasing intent
- **Local Service Quote Calculator** and **Lead Value ROI Calculator** target business revenue conversations directly, where buyers already have pain and spend intent.
- **Website Audit Scorecard** gives freelancers and agencies a repeatable diagnostic asset they can sell before design work, improving close rates.
- Browser games are engagement-focused and typically need heavier polish to monetize, while these tools are decision-support tools that support sales quickly.

### 2) Faster path to paid delivery
- Each of these deliverables is a complete mini-site with one clear outcome.
- They are lightweight static outputs with configurable copy and brand fields, so setup time is mostly data and text replacement.
- They can be sold immediately at different tiers (template + customization), which maps well to “order now” buyer behavior.

### 3) Lower support burden
- No backend, accounts, databases, or API keys required.
- If a buyer wants custom pricing rules/fields, changes are easy via HTML/CSS/JS only.
- Static behavior means no infrastructure failures and easier maintenance across platforms.

### 4) Clear value messaging
- Each gig maps to a practical business problem:
  - **Quote speed and consistency** (quote calculator)
  - **Ad budget confidence** (ROI calculator)
  - **Optimization prioritization** (scorecard)
- This framing aligns with urgent decision-making, unlike generic portfolio templates that require extra justification.

## Suggested sales channels
1. **Gumroad / Ko-fi / Payhip product bundles**
   - Offer downloadable zip package per gig + optional customization upgrade.
2. **Fiverr / Upwork gig pages**
   - Create one service listing per deliverable with fixed scope and add-ons.
3. **Instagram + LinkedIn outreach posts**
   - Post sample before/after screenshots and a concise “what it does in 60 seconds” video.
4. **Agency/SMB communities**
   - Offer first-party demos for service providers, growth coaches, and freelancers.

## First-offer positioning

Use one lead-in message:

> "I sell ready-to-use, no-backend mini-sites that help you close more clients in under one day of setup."

Suggested first pack ladder:

1. **Starter bundle (single tool)**
   - One mini-site built and brand-customized.
   - Price anchor: **$79–$149**
2. **Pack value bundle (any 2 tools)**
   - Includes one discovery walk-through call.
   - Price anchor: **$199–$349**
3. **Profit stack bundle (all 3 tools)**
   - Full setup + 2 revision rounds + handoff checklist.
   - Price anchor: **$449–$699**
